Powerful features, tailored to you

Everything you need to make AI-assisted coding portable, collaborative, and reliable — without sacrificing privacy.

Collaborative Sessions

Share sessions with teammates and friends using private or public links, with username-first social flows on compatible servers.

Expanded Provider Ecosystem

Use Claude Code, Codex, Gemini, OpenCode, Kilo, Kimi, Qwen, and Augment with one consistent session experience.

Multi-server Workflows

Use personal, team, and self-hosted servers side-by-side. Switch fast and keep accounts scoped per server.

Git-aware File Browser

Inspect changed files and real per-turn diffs in-session, with optional experimental Git operations when you want to act directly.

Steering + Pending Queue

For compatible backends, steer the active turn mid-run (manual or auto), or queue/edit/reorder/discard messages before execution.

Modes, Models, Permissions

Choose permission policy and, when supported, switch mode/model per session with clear apply timing (immediate or next-turn).

Notification-aware Routing

Jump to the exact session from a notification, including automatic server switching in multi-server setups.

Voice Options (Cloud, BYO, Local)

Use Happier Voice, your own ElevenLabs account, or local/device STT/TTS for flexible voice workflows.

Built-in Bug Reports

Report issues from app settings with optional diagnostics; server owners can disable bug report features when needed.

Comprehensive control of your coding sessions

Explore the workflow at a glance. Switch categories to see the product area and the feature explanation.

Sessions you can trust

Persistent sessions, reliable resume, and full history across devices — designed for long-running agent workflows.

Get Started in Minutes

Set up Happier and start coding from anywhere in just four simple steps.

1

Download the App

Get Happier from the App Store or Google Play on your mobile device.

2

Install the CLI

Run curl -fsSL https://happier.dev/install | bash on your computer (npm remains available as a fallback).

3

Pair Your Device

Scan the QR code in the app to securely connect your phone to your computer.

4

Start Coding

Run happier instead of claude and code from anywhere!

Terminal
# Instead of running claude directly:
$ claude
# Use happier to enable remote access:
$ happier
# Or for other providers:
$ happier codex
$ happier opencode
$ happier gemini
Fully Self-hostable

Own the stack. Stay independent.

Happier runs on your own machine — server, web UI, and daemon. You can use it as-is, audit it, or tailor it to your workflow.

One-command setup

Install and launch the full stack with hstack’s guided self-host profile.

Easy daily run

Start everything with a single command — the server, daemon, and UI gateway.

Secure remote access

Optional Tailscale integration gives you an HTTPS URL so your phone can connect securely from anywhere.

Terminal
# One-command setup (guided):
$ curl -fsSL https://happier.dev/self-host | bash
# Start Happier anytime:
$ hstack self-host status
# Optional: secure remote access for mobile:
$ hstack tailscale enable
$ hstack tailscale url

Prefer npm? Power users can still install with npm install -g @happier-dev/cli. Open hstack docs

Security First

Your Code is Your Code. Private. Always.

Happier is designed with privacy as a foundation, not an afterthought. We use the same encryption technology trusted by Signal.

End-to-End Encryption

Built using TweetNaCl, the same cryptographic library used by Signal. Your data is encrypted before it leaves your device.

Zero-Knowledge Architecture

Servers cannot read your data. Encryption keys never leave your devices. We literally cannot access your code.

Built in Switzerland

Developed in Switzerland with a strong focus on data protection, privacy laws, and developer transparency.

Built by Developers, for Developers

Happier is completely open-source. We built this because we wanted a more powerful, more social way to interact with AI agents — and we want to build it in the open, shaped by the people who actually use it.

Open Development

Fast Feedback Loops
